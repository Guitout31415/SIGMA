# CellExtractor

This bioinformatics pipeline automates the extraction of targeted cell types from multiple heterogeneous single-cell datasets (.h5ad, .rds, .csv). It is designed to simplify and accelerate integrative large-scale transcriptomic analyses.

## Installation
Clone the GitHub repository:
```
git clone https://github.com/Guitout31415/CellExtractor.git
cd CellExtractor
```
